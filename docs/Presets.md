## Preset Usage Tips

### Loading Presets
1. Click preset dropdown (top of GUI)
2. Browse categories
3. Click preset name to load
4. Fine-tune parameters to taste

### Saving User Presets
1. Adjust parameters
2. Click "Save Preset" button
3. Enter name and category
4. Preset saved to user preset folder

### A/B Comparison
1. Load preset into slot A
2. Adjust parameters
3. Click "A" button to recall original
4. Click "B" to hear your adjustments
5. Compare instantly!

### Preset File Location

**Windows:**
```
C:\Users\[YourName]\AppData\Roaming\BTZ Audio\BTZ\Presets\
```

**macOS:**
```
~/Library/Application Support/BTZ Audio/BTZ/Presets/
```

**Linux:**
```
~/.config/BTZ Audio/BTZ/Presets/
```

## Preset Development Notes

All factory presets follow these principles:
- **Gain-staged**: No huge loudness jumps between presets
- **Mono-compatible**: Low-end phase-coherent
- **CPU-efficient**: Oversampling only when needed
- **Genre-agnostic**: Work across multiple styles
- **Forward-compatible**: Will load in future versions

**Preset Format:** JUCE ValueTree (XML)
**Version:** 1.0.0
**Total Size:** ~200 KB (all 20 presets)
