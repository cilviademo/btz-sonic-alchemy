# /resume

Use this command to recover project context quickly in a new chat.

## Steps
1. Read:
   - `@.memory/00_PROJECT_OVERVIEW.md`
   - `@.memory/01_ARCHITECTURE.md`
   - `@.memory/02_DECISIONS_LOG.md`
   - latest file in `@.memory/sessions/`
2. Summarize:
   - current project goal
   - active constraints
   - recent decisions
   - current build/deploy status
3. Ask clarifying questions:
   - what should be prioritized now?
   - any architecture changes approved?
   - target environment (Windows/macOS, host DAW)?
4. Propose next steps:
   - short actionable plan (3-5 steps)
   - identify any blockers/dependencies

## Expected Output Template
- **Project state:** ...
- **Constraints:** ...
- **Recent decisions:** ...
- **Open issues:** ...
- **Questions:** ...
- **Proposed next steps:** ...
